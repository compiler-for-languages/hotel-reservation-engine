# Hotel Reservation Engine
Hotel Reservation Engine is a high-concurrency booking platform developed using Spring Boot, PostgreSQL, Redis, and React.

The system allows customers to search room availability, place booking requests, complete payments, and manage reservations while preventing overbooking through distributed locking mechanisms.

Key Features:
- JWT Authentication
- Room Availability Search
- Reservation Management
- Redis Distributed Locking
- Payment Workflow
- Admin Dashboard
- Real-Time Availability Tracking
